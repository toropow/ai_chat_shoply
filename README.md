# Консольный ChatGPT-бот для e-commerce

## Цель: собрать консольного бота поддержки магазина «Shoply», который:
* ведёт диалог с историей, отвечает кратко и по делу;
* использует внутренний FAQ (JSON) для типовых вопросов;
* по команде /order 12345 подтягивает статус из orders.json;
* логирует каждую сессию в logs/session_*.jsonl и считает использованные токены.
